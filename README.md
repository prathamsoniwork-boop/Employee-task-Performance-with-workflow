## 🎯 Project Overview

The Employee Task Performance and Workflow Management System is a comprehensive web-based application designed to streamline project management, task assignment, and performance tracking within organizations. It provides a centralized platform for companies to manage employees, projects, tasks, and monitor workflow efficiency.

---

## 🏗️ Technology Stack

### Frontend Technologies:
- **ASP.NET Web Forms 4.8.1** - Main web framework
- **HTML5/CSS3** - User interface styling
- **JavaScript** - Client-side interactions
- **AJAX** - Asynchronous data operations
- **Bootstrap** (implied) - Responsive design components

### Backend Technologies:
- **C# (.NET Framework 4.8.1)** - Server-side programming
- **SQL Server LocalDB** - Database management
- **ADO.NET** - Database connectivity
- **SQL Server Data Tools** - Database operations

### Data Visualization:
- **System.Web.DataVisualization** - Charts and graphs
- **Chart Controls** - Performance metrics visualization
